## Naming Convention

Files are named: `XXX-feature-name.md`
- `XXX` = Sequential number (001, 002, 003...)
- `feature-name` = Descriptive name (e.g., hero-slider, ci-cd-pipeline)

## Log Template

Each log file follows this structure:

```markdown

(Technical approach, libraries used, design choices)

(The actual prompt that triggered this implementation)

(List of files created/modified with code logic explanation)

(Notes for future modifications)
```
